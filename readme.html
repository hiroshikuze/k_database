機能詳細

[ 主なお勧めポイント / 主なマイナスポイント ]
顧客管理用のデータベースソフトです。

主なお勧めポイント

    　
  1. Web上で動くグループウェアとしても利用できます。
    なおPerl動作環境必須です。追加モジュールは不要です。
    2. 　
  2. 顧客情報と対応履歴の記録（追加※/削除※/編集）が可能です。
    ※の部分は管理者のみ行えます。
    3. 　
  3. 顧客情報および対応履歴の検索が出来ます。
    4. 　
  4. データベース以外の周りの表示部分のカスタマイズが容易です。
    主なレイアウトの編集は添付ファイル内の 『output_sub.pl』 の最後部分をいじるだけでOKです。
    5. 　
  5. 顧客リスト表示時に住所データがMapFan様のwebサイトと勝手に連携します。
    (…ですが、MapFan様の都合によりシステムが変更されれば、当然この連携は壊れますので、その際には各自で適当に修正する必要があります(^^;))。
    6. 　
  6. データはCSVテキスト（タブ区切り）形式で保存します。別ソフトとの連携が容易かも？
    7. 　
  7. ブラウザ上からExcelを起動させてデータの一覧を一応見る事が出来ます※。
    ※管理者のみ。要IE&Excel。
    8. 　
  8. データの追加・編集操作を行ったとき、データファイルのバックアップを１世代分のみ作成します。
  9. 起動時に管理者コメントが表示できます。
    データベース一般利用者に対して、連絡事項を伝えるのにご利用ください。
  10. 簡単なプログラム解説もれなくついてきます。自分用の用途に合うよう改造に役立ててください。
  11. [New!] ソフトの操作履歴が一部残ります / ソフト起動ファイル『index.cgi』のファイル名を変更する事ができます。
    セキュリティ向上に生かしてください。
  12. [New!] 登録した会社へ一斉にメールが送信できる※。
    ※管理者のみ。要sendmail、もしくは互換ソフト。
  13. 使用、再配布が無料。設置などによる料金請求も可。

主なマイナスポイント

  1. 項目部分や入力部分の変更が困難。
    友達から「デザインはどーでもいいから早く作ってくれないか？」と言われて作ったものなので、その辺りを深く練る事をしませんでした。
  2. 扱えるのはテキスト文のみである。
    デジカメで取った画像ファイル…などは扱えません。
  3. 端末を選ぶ。
    PC上で扱うことを前提に作成しています。携帯・PDA等からは利用できません、多分。
  4. サポートがイマイチ。
    私個人は顧客を管理する仕事にない、メインの職がプログラマーではない、他に作りたいものとかが色々ある、等の理由でバグ修正や機能追加は弱いです。
    なお、(資)ジョブサポート様のASPサービスでは、独自でこのソフトの各種サポートを行っています。
    不安な方はどうぞ。もう少しだけ詳しい詳細はこちら

バグ情報

顧客管理データベース ver 0.900E とそれ以前 ですが、重要なセキュリティホールが存在します。
最初の「最新順」（別に検索結果表示時でもいいのですが）を表示している時に外部サイトへのリンクをクリックしますとその際、環境変数のHTTP_REFERERによってデーターベースへのアドレスや暗号化状態のとはいえ管理者パスワードがバレ、第3者によってデータベースへ辿られてアクセスされたり管理者権限を取られる可能性があるのです。

ver 0.900F ではダミーのページを間に挟んで解決しましたので、以前のバージョンをご利用中の方はバージョンアップを是非お願いします。
